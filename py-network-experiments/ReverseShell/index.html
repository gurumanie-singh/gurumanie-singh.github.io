TCP Reverse Shell Utility

server.py listens for incoming connections, then launches an interactive shell over the TCP socket:

  • Uses subprocess.Popen to spawn a shell process.
  • Redirects stdin/stdout/stderr to the socket for remote command execution.

client.py connects and spawns a background thread to continuously read server responses, forwarding user keystrokes to execute commands remotely.
