TCP & UDP Socket Examples

Sockets.py houses modular functions demonstrating:

  • TCP server/client setup with socket.AF_INET, socket.SOCK_STREAM.
  • UDP datagram send/receive using socket.SOCK_DGRAM.
  • Blocking vs non-blocking modes and graceful shutdown handling.

Each function can be invoked via command-line arguments to spin up the desired example, providing a hands-on reference for socket basics.
