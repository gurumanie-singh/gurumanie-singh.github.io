COMS309: Software Development Practices

In COMS309, I collaborated in a four-person team to build a full-stack job-search web application, designed to simplify how students discover and apply for on-campus positions. We followed Agile processes—conducting stakeholder interviews, crafting user stories, and iterating through sprints to deliver features on schedule.

As lead backend developer, I architected the server logic using Java, designed a normalized MySQL database schema, and implemented RESTful APIs for user authentication, job posting, search filters, and application tracking. I wrote comprehensive unit and integration tests, set up continuous integration workflows, and monitored performance metrics to ensure scalability and reliability.

This hands-on experience honed my skills in requirements analysis, object-oriented design, backend integration, and collaborative development—preparing me for complex software engineering challenges.
